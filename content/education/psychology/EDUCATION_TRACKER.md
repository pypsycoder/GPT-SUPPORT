## UI структура раздела «Обучение»

```
Обучение
├── 🧠 Психологическое сопровождение   [9 уроков] — свободный порядок
└── 🫀 Жизнь с диализом                [9 уроков] — свободный порядок
```

**Решения:**
- Порядок прохождения: свободный (на пилоте смотрим что читают первым и где бросают)
- Прогресс: считается отдельно по каждому блоку
- Бейдж: за завершение каждого блока отдельно

## Правила контента (зафиксированные в процессе)

1. Паттерн «Это не X — это Y» — не чаще 1 раза на весь урок
2. «Замечать X — уже Y» — избегать в anchor
3. «Маленький шаг» — только в мотивации как концептуальный термин
4. «Это навык» — не повторять в каждом модуле
5. «Просто без названия» — не использовать (было в двух модулях одинаково)
6. Реальные мысли пациента вместо «характер/слабость»: «другие справляются», «что со мной не так»
7. Якоря — только на существующие разделы платформы
8. HADS не везде — ТОБОЛ для адаптации/выгорания, Распорядок для действий
