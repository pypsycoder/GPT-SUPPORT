## 2. Формула подсчёта

```
Сырой балл субшкалы = сумма ответов по пунктам субшкалы (каждый 0–3)

Нормализованный балл (для сравнения субшкал между собой):
  normalized = (raw / max_possible) × 100

Пример:
  Бегство-избегание: сырой = 14, макс = 24
  normalized = 14/24 × 100 = 58.3
```

## 4. Интерпретация профиля

### Адаптивный vs дезадаптивный репертуар

```python
adaptive_subscales     = [самоконтроль, поиск_поддержки, планирование, переоценка]
maladaptive_subscales  = [конфронтация, избегание]
mixed_subscales        = [дистанцирование, ответственность]

# Нормализованные баллы (0–100)
adaptive_mean     = mean(adaptive_normalized)
maladaptive_mean  = mean(maladaptive_normalized)

adaptive_ratio = adaptive_mean / (adaptive_mean + maladaptive_mean)
# 0.0 = полностью дезадаптивный
# 1.0 = полностью адаптивный
```

### Пороги adaptive_ratio для ICF d240

| adaptive_ratio | ICF-определитель | Интерпретация |
|---|---|---|
| ≥ 0.65 | 0 — нет нарушения | Преобладает адаптивный копинг |
| 0.45–0.64 | 2 — умеренное нарушение | Смешанный репертуар |
| < 0.45 | 3 — тяжёлое нарушение | Преобладает дезадаптивный копинг |

> Пороги операциональные (не валидированы на ГД-выборке) — фиксировать в диссертации как допущение.

## 7. Динамический анализ (T0 → T1, для диссертации)

Основной показатель эффекта вмешательства:

```
Δ adaptive_ratio = adaptive_ratio(T1) - adaptive_ratio(T0)

Δ > 0  → сдвиг в сторону адаптивного копинга (ожидаемый эффект)
Δ < 0  → сдвиг в сторону дезадаптивного (нежелательный)
Δ ≈ 0  → нет изменений
```

Дополнительно — попарные сравнения по каждой субшкале (Wilcoxon signed-rank, т.к. данные ординальные).
