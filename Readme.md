# Основные модули проекта

* **config.py, .env, requirements.txt** — глобальные настройки приложения, зависимости и переменные окружения.
* **core/db/** — асинхронный движок SQLAlchemy и фабрика сессий (engine.py, session.py).
* **app/models/** — базовые модели проекта (общий Base) и модели пользователей (users).
* **app/users/** — модули с моделями и CRUD-операциями для управления данными пользователей.
* **app/scales/** — реализация масштабов опросов:
* **schemas/hads.py** — декларативное описание шкалы HADS (вопросы, варианты ответов, пороговые значения, интерпретации).
* **models.py** — модели ScaleDraft и ScaleResponse с JSON-полями для хранения ответов и результатов.
* **crud.py** — асинхронные функции для создания черновиков, записи ответов и финализации результатов.
* **fsm/questionnaire.py** — универсальный FSM-роутер на aiogram для проведения опросов.
* **bots/TG_bot/** — реализация Telegram-бота:
* **main.py** — точка входа.
* **handlers/** — обработчики команд и FSM для опросов.
* **keyboards/** — генерация клавиатур для взаимодействия.
* **bots/shared/utils.py** — общие утилиты, например, логирование.
